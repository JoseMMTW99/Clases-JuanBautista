¿Qué es la Consola y Para Qué Sirve?

La consola es una herramienta esencial para los desarrolladores. Nos permite ver información sobre el código en ejecución, errores y resultados sin afectar la interfaz de usuario.

Para abrir la consola en tu navegador:

  • En Chrome o Firefox: Haz clic derecho en la página y selecciona "Inspeccionar", luego ve a la pestaña "Consola".
  • También puedes presionar F12 o Ctrl+Shift+I (o Cmd+Option+I en Mac) para abrir directamente las herramientas de desarrollo.
1. Mostrar Mensajes en la Consola

Usamos console.log() para mostrar mensajes o resultados en la consola. Esto es útil para depurar y verificar el comportamiento de nuestro código. 

console.log("Hola, esto es un mensaje en la consola."); // Muestra un mensaje de texto en la consola
let numero = 42;
console.log("El valor de la variable 'numero' es:", numero); // Muestra el valor de una variable en la consola

Recuerda, los mensajes en la consola no se muestran en la página web, solo en las herramientas de desarrollo del navegador.

2. Errores en la Consola

Cuando hay un error en el código, JavaScript lo mostrará en la consola. Esto nos ayuda a identificar y solucionar problemas en nuestro código. 

console.log(mensajeQueNoExiste); // Esto dará un error, ya que 'mensajeQueNoExiste' no está definida

El error se verá algo así en la consola: "Uncaught ReferenceError: mensajeQueNoExiste is not defined". Esto indica que estamos tratando de usar algo que JavaScript no reconoce.

3. Otros Comandos de Consola

Además de console.log(), hay otros comandos útiles:

  • console.error(): Muestra errores en rojo.
  • console.warn(): Muestra advertencias en amarillo.
  • console.table(): Muestra datos en forma de tabla.
console.error("Este es un mensaje de error"); // Muestra un error en la consola
console.warn("Este es un mensaje de advertencia"); // Muestra una advertencia en la consola

let personas = [
  { nombre: "Juan", edad: 14 },
  { nombre: "Ana", edad: 13 },
  { nombre: "Luis", edad: 15 }
];

console.table(personas); // Muestra los datos en una tabla en la consola

La consola es una herramienta indispensable para los desarrolladores. Nos permite verificar el funcionamiento del código, ver resultados y encontrar errores sin afectar la vista del usuario en la página web.
