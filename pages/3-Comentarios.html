Comentarios en JavaScript

Los comentarios en JavaScript son útiles para hacer notas y explicaciones dentro del código sin que este se ejecute. Pueden ser de una sola línea o de varias líneas.

1. Comentarios de Una Línea

Los comentarios de una línea se hacen usando //. Todo lo que esté después de // en esa línea será un comentario. 

// Esto es un comentario de una sola línea
let nombre = "Juan"; // También se puede usar después de una línea de código
2. Comentarios de Varias Líneas

Para escribir comentarios largos o de varias líneas, usamos /* al inicio y */ al final. Todo lo que esté entre estos símbolos será ignorado por JavaScript. 

/* 
Este es un comentario de varias líneas.
Aquí estamos guardando la edad 14 en una variable llamada 'edad'.
*/
let edad = 14;
3. Cuándo Usar Comentarios

Los comentarios deben usarse para:

  • Explicar por qué se está haciendo algo, en lugar de qué se está haciendo.
  • Proveer contexto o aclaraciones cuando el código puede no ser claro para otros.
// Este código convierte una temperatura en grados Fahrenheit a Celsius
let celsius = (fahrenheit - 32) * 5/9;

Los comentarios son una herramienta poderosa para documentar el código, haciendo que sea más fácil de entender tanto para ti como para otros desarrolladores.
